# Weekly Review

Generate a summary of my work from the past 7-10 days.

## Steps

1. Get my git email: `git config user.email`
   - If not found, ask me to set it
2. Go through all my commits on the main branch for the past 7-10 days where I am the author
3. Also check other branches I've worked on in that period

## Output Format

### What I Shipped
2-5 bullet points summarizing the changes.

### Breakdown
- **New Features:** net-new functionality
- **Bug Fixes:** issues resolved
- **Tech Debt:** refactoring, cleanup, improvements
- **Infra/CI:** deployment, pipeline, config changes

### Highlight
Short paragraph connecting the work to business impact (People Finder adoption, platform stability, team velocity).
